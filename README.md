# Student Study Planner

## Project Description
The Student Study Planner is a web-based application designed to help students organize their study schedules efficiently. It allows users to plan tasks, manage time, and track their daily study activities in a simple and user-friendly interface.

This project is developed as part of the User Interface Design (UID) course and focuses on creating an intuitive and responsive UI for better user experience.

## Features
- Add and manage study tasks
- Set priorities and deadlines
- Mark tasks as completed
- Clean and simple user interface
- Responsive design for different devices

## Technologies Used
- HTML5
- CSS3
- JavaScript
